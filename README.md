SuperHero Viewer
This app was built uising Ionic 7, Angular, and Capacitor. The app allows users to view superhero characters, save favourites, and toggle dark mode.
I created this for my Front-End Project for college.

Features
Connects to an external AI to browse hundreds of characters
View hero statistics
Add/remove Favourites
Search for heroes
Dark/light mode toggle
Vibration
Ionic UI
Angular Router

Technology Used
Ionic 
Angular 
Capacitor
Ionic Storage
HttpClient + Observables
SuperHero API
